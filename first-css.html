Types of tags in HTML

Go to bottom

Container Tags

Container tags help structure the webpage by dividing it into distinct sections for better design and functionality. These tags which don't have their own content eg.,
  • table
  • body
  • head
  • div

Standalone Tags

Standalone tags in HTML, also known as self-closing tags, do not require a closing tag. They automatically end after their attributes are defined
  • br
  • img
  • hr

Content Tags

Content tags are used to structure and display the content of a webpage.Hierarchy of the information which must be opened and closed eg.,
  • h1
  • td
  • p
  • b
  • i

Parent Tags

Parent tags contain one or more child tags, creating a nested structure that defines the document's hierarchy eg.,
  • table
  • tr
  • td

Attribute Tags

Tags with attributes provide additional information or modify the behavior of an element eg.,
  • id
  • class
  • style
